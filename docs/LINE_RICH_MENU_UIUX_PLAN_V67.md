# LINE Rich Menu UI/UX Plan V67

## 版本定位
- 單頁 Rich Menu。
- 採用科技儀表板邏輯。
- 標題橫幅移至底部。
- 左側提供系統說明、熱門快捷功能、狀態資訊。
- 右側以大卡片提供主要功能，提升手機使用者直覺點擊性。
- 不分頁、不上下掀頁。

## 點擊區
- 外勤回報
- 成果查詢
- 即時機況
- 設備管理
- 案件追蹤
- 月報統計
- 法規中心
- 百大熱點
- 陳情趨勢
- 新聞快訊
- 左側快捷提示區也對應常用功能

## 部署
部署後至 `/admin.html` 點選「一鍵更新」。
系統會強制刷新 Rich Menu，並同步 Google Sheet。
