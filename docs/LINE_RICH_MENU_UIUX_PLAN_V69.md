# LINE Rich Menu UI/UX Plan V69

## 更新重點
- 使用新的明亮科技卡片式 Rich Menu。
- 不使用左上角系統小幫手區塊。
- 單頁 Rich Menu，不分頁、不上下掀頁。
- 右下角隱藏區連續點擊三次後，LINE Bot 會提供後端平台入口。
- 後端平台入口需輸入密碼，預設 `69677323`，也可用 `ADMIN_PASSWORD` 覆蓋。
- 保留強制刷新 Rich Menu 與 Google Sheet 自動同步。

## 隱藏功能
1. 在 Rich Menu 右下角九宮格 / 設定區連點三下。
2. Bot 回覆後端平台入口。
3. 開啟後輸入密碼。
4. 通過後進入 `/admin.html`。
